🚀 Faculty Management System (MERN Stack)

Developed and customized a full-stack Faculty Management System using MongoDB, Express.js, React.js, and Node.js. The application provides role-based access for Admin, Faculty, and Students, enabling efficient management of academic activities, timetables, notices, subjects, and student records. Focused on enhancing functionality, improving the user interface, optimizing database operations, and implementing secure authentication to deliver a seamless user experience.
